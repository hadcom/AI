# AI

## Inventor VBA Rule

This repository contains an example VBA rule for Autodesk Inventor. The file `InventorVBARule.bas` demonstrates how to change the `Length` parameter in the active document to `150 mm` and update the model.

To use this rule:
1. Open Inventor and the document you want to modify.
2. Open the VBA editor (`Alt+F11`).
3. Import the `InventorVBARule.bas` file.
4. Run `SetLengthParameter`.
